| 标签         | 含义                                   | 示例                             |
| :----------- | :------------------------------------- | :------------------------------- |
| `@file`      | 指定当前文件名（文件注释必须包含此项） | `@file main.c`                   |
| `@brief`     | 简短描述（一句话概括文件作用）         | `@brief 核心控制逻辑`            |
| `@details`   | 详细描述（可多行，补充说明）           | `@details 包含了系统的主循环...` |
| `@author`    | 作者姓名                               | `@author 张三`                   |
| `@version`   | 文件/模块版本号                        | `@version 1.2.0`                 |
| `@date`      | 创建或最后修改日期                     | `@date 2024-05-20`               |
| `@copyright` | 版权信息                               | `@copyright MIT License`         |

| 标签                 | 含义                               | 示例                             |
| :------------------- | :--------------------------------- | :------------------------------- |
| `@brief`             | 函数功能简述                       | `@brief 计算两个数的和`          |
| `@details`           | 函数功能详述（算法细节等）         | `@details 使用快速傅里叶变换...` |
| `@param`             | 参数说明（后跟参数名和描述）       | `@param a 第一个加数`            |
| `@return`            | 返回值整体说明                     | `@return 返回计算结果`           |
| `@retval`            | 具体返回值说明（后跟具体值和描述） | `@retval 0 成功`                 |
| `@note`              | 注意事项/补充说明                  | `@note 此函数不是线程安全的`     |
| `@warning`           | 警告信息（潜在危险）               | `@warning 调用前必须先初始化`    |
| `@see`               | 相关引用（跳转到其他函数/文件）    | `@see init_system()`             |
| `@code` / `@endcode` | 代码示例块（包裹示例代码）         | 见上方代码示例                   |

| 修饰符           | 含义                     | 适用场景                                 |
| :--------------- | :----------------------- | :--------------------------------------- |
| `@param[in]`     | 输入参数（只读）         | 函数只读取该值，不修改它（如 `int a`）   |
| `@param[out]`    | 输出参数（函数会修改它） | 指针参数，用于将结果传回给调用者         |
| `@param[in,out]` | 输入且输出               | 指针参数，函数既读取它的初始值，又修改它 |

| 标签      | 含义             | 示例                          |
| :-------- | :--------------- | :---------------------------- |
| `@struct` | 结构体说明       | `@struct User 用户信息结构体` |
| `@enum`   | 枚举类型说明     | `@enum Status 系统状态枚举`   |
| `@def`    | 宏定义说明       | `@def PI 圆周率常量`          |
| `///<`    | 成员变量行尾注释 | `int age; ///< 用户年龄`      |
